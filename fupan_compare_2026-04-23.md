# 对比结论
- 市场环境从 4月22日的“指数偏强、午后修复、科技主线强化”，切换为 4月23日早盘的“高成交、弱宽度、指数回落、强分化”。

- 核心个股结构从“趋势核心继续新高”转为“高位科技权重兑现、低位事件题材接力”。中际旭创由趋势中军转为回落观察锚，天孚通信新增为负反馈锚。
- 情绪结构从“修复延续中的结构强化”降级为“局部强、整体弱”。金螳螂五连板超预期，但商业航天板块仍未形成合力。

# 延续与变化

4月22日市场的核心判断是：指数环境偏强，资金集中在 AI硬件、通信、算力、芯片等高景气科技链，情绪处于“修复中后段但未全面高潮”。4月23日早盘则显示，成交继续放大，但放量没有带来普涨，反而伴随指数走弱、下跌家数维持高位，说明资金从集中抱团转向兑现与轮动。

被延续的判断主要有三点。第一，市场仍不是普涨环境，两日都强调个股分化和资金集中。第二，商业航天仍是“个股强、板块弱”，金螳螂继续晋级，但板块仍净流出、跟随不足。第三，高位负反馈仍是重要风险，上一日圣阳股份一字跌停，当前日则表现为天孚通信大跌、锂电池西藏珠峰跌停、海科新源大跌。

被证伪或明显弱化的判断，主要集中在 AI硬件/通信。上一日该方向是最强主线，通信净流入最大，东山精密、长飞光纤、新易盛、中际旭创等共同强化；当前日 AI硬件/通信转为跌幅 -1.55%、净流出 182.49 亿，中际旭创创新高后回落，天孚通信成为负反馈锚，说明原主线已从“趋势主升”进入“高位分歧与兑现”阶段。

超预期变化有两处。第一，金螳螂从四连板继续晋级到五连板并回封，短线高度超预期，但仍未带动商业航天板块全面转强。第二，CPU 方向成为科技线内部新的承接点，综艺股份涨停、龙芯中科和海光信息走强，显示资金并未完全撤离科技，而是从 AI硬件高位核心向低位科技分支切换。

# 对下一交易日的启发

1. 重点看 AI硬件/通信能否止跌修复。中际旭创、天孚通信、新易盛若不能重新稳住，说明 4月22日的最强主线可能进入阶段性退潮。

2. 观察 CPU 是否从“承接分支”升级为新主线。综艺股份能否晋级、龙芯中科和海光信息能否继续强于大盘，是判断科技资金是否完成高低切的关键。

3. 金螳螂仍是短线情绪总锚。若继续超预期且断板无强负反馈，局部连板情绪仍可维持；若高标断板并拖累商业航天，短线情绪会进一步降温。

4. 燃气轮机只看开板后的换手承接。水发燃气、福鞍股份若仅一字加速而无板块扩散，容易停留在事件套利；若换手后仍强，才可能提高主线级别。

5. 负反馈线索要盯紧锂电池和高位科技。西藏珠峰、海科新源、天孚通信若继续扩散亏钱效应，会压制赛道和高位趋势股的整体风险偏好。
