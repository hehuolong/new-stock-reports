# 市场总览

2026年4月17日早盘，三大指数明显分化：上证指数水下震荡，创业板指一度涨近1%。成交端维持高热，9:42两市成交突破5000亿，10:12突破1万亿，11:16突破1.5万亿，按直播口径预计全天成交在**2.4万亿至2.5万亿**，较前一交易日**放量6%至9%**。  
但量能扩张并未转化为普涨结构，10:06下跌个股接近4000家，11:30仍有**超3500家下跌**，说明市场是典型的“**指数分化、权重抱团、个股普遍承压**”格局。

涨停家数方面，结构化数据**未提供全市场涨停总数**，只能确认局部方向涨停活跃，例如通信方向“近10家个股涨停”，AI硬件、芯片、算力均有多只涨停股出现。  
情绪上，适合定义为“**局部强势、整体分化**”：高位连板仍在推进，圣阳股份7连板、博云新材5连板、利通电子4天3板，但真视通炸板、微盘股走弱，说明短线资金更偏向高辨识度核心资产与少数龙头，而非全面扩散。

# 主线与轮动

## 最强主线

最强主线仍是**AI硬件/通信链条**，且是今天最具“业绩+趋势+资金”共振特征的方向。核心催化来自**中际旭创一季报高增长**，直接强化了光模块、PCB、连接材料等细分的景气预期。板块信号显示，通信板块涨幅**1.27%**、净流入约**103.46亿元**，是已披露方向中资金承接最强的一档。

盘面特征有三点：

1. 竞价即强，天通股份大单一字，中际旭创、东山精密、新易盛高开。
2. 趋势中军持续新高，中际旭创、新易盛、东山精密再创历史新高，胜宏科技逼近涨停。
3. 首板和补涨扩散同步出现，剑桥科技、光迅科技、广合科技、东材科技相继涨停。

这意味着今天的主线并非纯情绪博弈，而是**机构趋势资金主导，游资做补涨扩散**。从交易视角看，这类行情的核心不在低位普涨，而在于中军能否持续抬升估值斜率。

# 情绪判断

当前情绪阶段更接近“**主线强化期中的分化交易**”，而非全面主升。

| 维度 | 判断 | 说明 |
| --- | --- | --- |
| 连板高度 | 偏强 | 圣阳股份7连板，博云新材5连板，利通电子4天3板 |
| 板块合力 | 主线强，支线一般 | 通信最强，芯片次之，算力更多靠高标维持 |
| 容错率 | 一般偏低 | 真视通炸板，欧莱新材、锂电池分支有冲高回落 |
| 风格偏好 | 大票/中军优先 | 中际旭创、新易盛、东山精密、胜宏科技显著强于多数个股 |
| 市场广度 | 偏弱 | 下跌家数超3500家，微盘股走弱 |

短线资金偏好已经很明确：

1. 偏好有业绩支撑、趋势明确的科技权重。
2. 偏好有辨识度的高位龙头，而不是低位泛题材。
3. 对后排跟风和弱逻辑分支容忍度下降。

因此，今天市场的核心矛盾并不是“有没有机会”，而是“**机会高度集中在少数核心资产，非核心交易难度显著提高**”。
