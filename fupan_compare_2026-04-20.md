# 延续与变化

## 市场环境变化

4月17日市场是典型的“指数修复但个股分化”。虽然成交显著放大，全天预计超2.4万亿，但早盘下跌个股一度接近4000家，午后才修复至上涨个股超2000家，赚钱效应集中在通信、芯片、算力等少数方向。

4月20日市场环境进一步改善，13:40两市成交已突破2万亿，预计全天超2.6万亿，较上一交易日继续放量约8%。更重要的是，早盘上涨个股一度超3000家，说明市场宽度较上一日明显改善。不过午后圣阳股份板上放量、凯华材料冲高回落、宝丽迪大幅回落，显示环境虽更活跃，但并非无风险普涨。

结论：上一日“成交放大、结构性行情”的判断被延续；“广度不足”在当前早盘阶段得到修复，但午后分化说明宽度改善并未转化为全面主升。

## 最强主线变化

4月17日最强主线是 `AI硬件/通信`。核心逻辑是中际旭创一季报高增长带动光模块、PCB、AI硬件链条，形成“业绩驱动 + 趋势加强”的主线结构。中际旭创、新易盛、天通股份、剑桥科技构成当日主线强度链。

4月20日最强主线切换为 `商业航天`。商业航天板块涨幅1.63%，净流入约67.05亿元，并出现“超10家个股涨停”。其结构比上一日更完整，从博云新材六连板的情绪高度，到中国卫星、中国卫通等容量中军，再到神剑股份、广联航空、天银机电、西部材料等弹性扩散，形成高标、权重、补涨三层共振。

结论：商业航天从上一日支线升级为当前主线，属于超预期变化；AI硬件/通信没有退潮，但从最强主线降为强势延续支线。

## 核心个股与带动关系变化

上一日核心带动链条是：

`中际旭创业绩超预期 -> 通信/AI硬件权重走强 -> 剑桥科技、光迅科技、广合科技等涨停扩散 -> 指数与情绪获得支撑`

当前核心带动链条变为：

`博云新材六连板 -> 商业航天扩散 -> 中国卫星、中国卫通等权重确认 -> 神剑股份、广联航空、天银机电等弹性票强化`

这说明市场定价重点发生变化：上一日资金更偏“业绩确定性 + 趋势中军”，当前更偏“政策催化 + 板块扩散 + 容量确认”。

## 情绪结构变化

4月17日情绪定义为“高位抱团延续、广度不足、轮动加速”，不是全面主升，也不是退潮。

4月20日情绪定义为“强势延续后的分歧初期”。圣阳股份8连板、博云新材6连板、沃格光电8天5板，说明高度继续打开；但品高股份炸板、凯华材料回落、宝丽迪回落、圣阳股份板上放量，说明高位筹码开始松动。

结论：高位抱团延续的判断被验证，但“高位分化日”进一步演化为“分歧初期”。风险没有扩散成指数级退潮，但短线容错率开始下降。

# 对下一交易日的启发

1. 继续跟踪商业航天能否从日内最强题材升级为阶段主线。关键不只看博云新材是否继续晋级，还要看中国卫星、中国卫通、航天电子等容量票是否继续承接，以及神剑股份、广联航空、天银机电、西部材料等弹性票是否保持队形。

2. 重点观察圣阳股份的放量后承接。它已经从7连板晋级8连板，延续了上一日高位抱团判断，但当前板上放量说明分歧加大。若继续强封，可支撑连板高度；若出现明显负反馈，可能影响算力、AI硬件及高位接力情绪。
