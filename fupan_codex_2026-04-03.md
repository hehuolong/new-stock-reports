# 市场总览

4月3日早盘，指数走出明显的**高开回落**。09:29时三大指数集体高开，创业板指一度涨超0.5%，但到09:43沪深主板已相继翻绿，10:09上证指数跌超0.5%、微盘股指跌超3%，11:30上证指数跌近1%。量能方面，09:56两市成交突破5000亿、同比缩量15%；11:12成交突破1万亿，按直播口径预计全天**不足1.7万亿**，较上个交易日缩量约11%-12%。

从市场宽度看，弱势特征非常明确。09:56统计显示，**上涨789家、下跌4343家、涨停7家、跌停1家**；至11:30，直播摘要已明确“**下跌个股近4500家**”“**联环药业等超10家个股跌停**”。这意味着市场从开盘分歧迅速转向普跌，且亏钱效应在盘中继续扩散。由于数据中未提供午间或收盘口径的涨停总家数，**涨停家数只能引用09:56时点的7家**，不宜外推为全天结论。

整体情绪判断是：**指数弱、宽度差、题材活口集中、短线情绪分化偏退潮**。盘面并非没有进攻方向，但更多表现为少数核心票和局部板块抗跌，绝大多数冲高题材承接不足。

# 主线与轮动

## 最强主线

### 1. AI硬件 / 通信
这是早盘最有持续性的方向，也是少数兼具**板块净流入、趋势中军、连板前锋**的主线。

板块层面，通信板块净流入**147.86亿**，显著强于其他方向；个股层面，**汇源通信（000586）二连板**负责短线辨识度，**长飞光纤（601869）再创历史新高**负责趋势定价，**德科立（688205）20cm涨停**、**腾景科技（688195）**、**光库科技（300620）**共同强化板块弹性。驱动上，直播给出的明确催化是**欧洲裸光纤价格3月大幅上涨**，属于价格逻辑与AI硬件景气逻辑叠加。

交易视角看，这条线之所以成为主线，不在于板块涨幅有多高，而在于在指数走弱、市场普跌时仍能维持**强个股不断强化、趋势股持续创新高、资金净流入居前**。这是一种典型的“**弱市抱团主线**”特征。

### 2. 算力 / 国产算力 / GPU
算力和GPU更像AI硬件主线向内延伸的分支。09:57思特奇（300608）20cm涨停，东方国信（300166）涨超8%；11:10国产算力再异动，摩尔线程（688795）、沐曦股份（688802）、寒武纪（688256）跟涨。板块净流入上，**算力+29.66亿、GPU+11.70亿**，资金偏好仍然偏向科技成长链。

但需要看到，算力和GPU板块自身涨跌幅分别为**-0.96%**、**-0.22%**，说明板块内部并非全面走强，而是**资金集中打辨识度个股**。因此它更适合作为主线强化的补涨分支，而非独立主升浪。

## 轮动支线

### 1. 医药
医药的看点主要集中在**高标连板**，而不是板块整体强度。**津药药业（600488）六连板**、**重药控股（000950）二连板**维持了前排热度，但板块涨跌幅为**-2.45%**、净流入为**-27.10亿**，和“强板块”并不匹配。也就是说，医药更接近**情绪抱团支线**，核心票强，板块并不强。

### 2. 跨境支付
该方向竞价最强，09:26就有**中油资本（000617）**、**翠微股份（603123）**大单一字，**贝肯能源（002828）**一字二连板，且有“人民币结算首现霍尔木兹海峡”的题材解读支撑。但盘中09:40即出现**冲高回落、相继炸板**，虽然随后翠微股份回封、中油资本反复开板，但强度明显弱于竞价预期。

这类走势说明，跨境支付早盘更像**消息驱动的一致性抢筹**，随后遭遇兑现，资金态度偏短炒，不具备稳定承接。板块净流入**18.92亿**不低，但板块涨跌幅仅**-0.39%**，也印证了“有资金进、但分歧极大”。

### 3. 航运、电子气体、化工
这几个方向都出现了轮动异动，但持续性和板块共振不足。

- 航运：11:20异动，招商南油（601975）逼近涨停，但板块涨跌幅仍为**-1.08%**，更像防御性或局部事件催化轮动。
- 电子气体：中船特气（688146）涨超12%，但板块净流入仅**369万**，力度偏弱，难言主升。
- 化工：苏利股份（603585）涨停，但板块整体**-2.51%**且净流出**21.42亿**，典型个股行情，板块无跟随。

# 情绪判断

当前情绪不是冰点，但已经从“可进攻”切换到“**强分化、弱容错**”阶段。

| 维度 | 表现 | 判断 |
|---|---|---|
| 连板高度 | 津药药业六连板，新能泰山九天七板 | 高度仍在，说明情绪未彻底崩 |
| 炸板回落 | 星辉环材、通达股份、新中港 | 接力风险显著上升 |
| 跌停扩散 | 联环药业等超10家跌停 | 亏钱效应已扩散 |
| 宽度 | 下跌近4500家 | 普跌环境确认 |
| 主线强度 | AI硬件相对独立走强 | 存在结构性机会 |
| 容错率 | 一致性题材冲高回落 | 不适合盲目追高 |

核心矛盾在于：**指数和市场宽度持续走弱，但局部主线和高辨识度个股仍在硬顶**。这会导致资金进一步向少数确定性方向集中，形成“**越弱越抱团**”的交易结构。

资金偏好也比较清晰：

1. 偏好有产业逻辑、能被趋势资金接受的科技硬件方向，尤其是通信/光纤/算力。
2. 偏好有辨识度的核心票，不愿在后排扩散上承担风险。
3. 对消息刺激型一致性题材，更多采取快进快出，而非持续抱团。
4. 对微盘、后排补涨、炸板反抽容忍度明显下降。
