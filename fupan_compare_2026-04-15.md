# 对比结论
- 市场环境从4月14日的“高成交下主线分化、尾盘权重修复”切换到4月15日的“高开后持续走弱、局部抱团防守”，指数承载题材的能力明显下降。
- 最强主线由4月14日的AI科技链主导，转为4月15日医药成为日内最强主攻方向，算力仍有强度，但已从“板块主升”退化为“高辨识度个股抱团”。
- 核心赚钱效应从“算力情绪票 + 存储趋势票 + AI硬件容量票”三维共振，收缩为“医药梯队 + 算力前排连板/弹性票”两类核心，科技权重和中军明显掉队。
- 4月14日尾盘异动的锂电池，4月15日被直接证伪，成为最明确负反馈方向；4月14日提示的商业航天延续了活跃度，但仍未升级为绝对主线。
- 情绪结构由“高位分歧但整体偏强震荡”，进一步演化为“高位抱团、中位掉队、后排承接弱”，操作重心更集中于少数辨识度核心。

# 延续与变化
**市场环境变化**

4月14日的核心特征是成交放大、指数偏强、午后分歧、尾盘靠寒武纪/宁德时代/中际旭创等权重修复；4月15日虽然成交预估继续维持在2.4万亿至2.5万亿高位，但指数表现明显更弱，深成指和创业板指午后跌幅扩大，说明高流动性并没有带来普涨，反而强化了高频轮动和局部抱团。

**最强主线变化**

4月14日最强主线明确是AI科技链，内部以算力、存储、AI硬件为核心；4月15日则变成医药最强，且医药具备“政策催化 + 板块持续强化 + 梯队完整”的特征。算力没有退场，但角色从“主攻主线”降为“高辨识度局部强势方向”；AI硬件则从前一日的重要主线分支，进一步暴露出大票承接不足的问题。

**情绪结构变化**

4月14日情绪是“修复强化 -> 高位分歧 -> 权重托底”，仍属偏强震荡；4月15日则是“高位能晋级，但中位和后排容错率下降”，更接近抱团防守。也就是说，情绪没有全面退潮，但扩散性明显变差。

**哪些判断被延续**

- “科技主线仍强但高位分歧明显”被延续。4月15日算力前排和弹性票继续强，但中际旭创、新易盛、沪电股份等权重继续转弱，验证了4月14日对科技内部高低切的判断。
- “商业航天需要看次日承接”被部分延续。4月15日商业航天继续活跃，航天工程二连板、神剑股份涨停创历史新高，说明并未一日游。
- “市场更偏好高辨识度龙头与容量中军，而非纯低位跟风”被延续。4月15日同样体现为核心股有溢价、后排承接弱。

**哪些判断被证伪**

- 4月14日尾盘锂电池被定义为可能的切换尝试，4月15日被明确证伪，海科新源大跌、孚日股份跌停、板块大额净流出，属于最典型的隔日失败轮动。
- 4月14日存储被视为科技线中最稳趋势分支，但4月15日复盘中存储已不再占据核心叙事，说明其至少在当日失去主导地位，未延续前一日的主线核心地位。

**哪些出现超预期变化**

- 医药成为最强主线，属于超预期变化。4月14日医药并非主要观察方向，4月15日却形成最完整的主攻梯队。
- 科技权重大票的承压程度超预期。4月14日还是“午后分歧、尾盘修复”，4月15日则演化为通信板块大额净流出、AI硬件权重午后继续走弱。
- 智能电网承接走强，也属于新增的有效轮动线索，强于4月14日报告中的地产、锂电等午后套利方向。
