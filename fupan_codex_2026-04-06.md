# 市场总览

2026年4月3日，市场呈现典型的**高开低走、缩量转弱**格局。上证指数收跌 **1%**，再度失守 **3900点**；两市成交额 **不足1.7万亿**，较前一交易日**缩量约10%**。市场宽度明显恶化，**上涨698家、下跌4459家**，涨停 **36家**、跌停 **24家**，亏钱效应显著强于局部赚钱效应。

从短线交易视角看，指数、量能、个股普跌三项同时走弱，说明当天并不是全面进攻环境，而是**弱市中的局部抱团与题材轮动**。情绪端没有给出结构化指标，但结合连板晋级与炸板、跌停扩散情况，可判断为**分歧加剧中的弱修复失败**。

# 主线与轮动

## 最强主线

### 1. AI硬件 / 通信
这是全天最有辨识度、也是资金相对最认可的方向。板块层面，**通信板块净流入约145.31亿元**，在已给数据中最强；盘中表现也最持续，早盘即有**汇源通信（000586）二连板**，随后**德科立（688205）20cm涨停**、**长飞光纤（601869）再创历史新高**、**亨通光电（600487）盘中触板**，形成了从情绪票到趋势中军再到20cm弹性的完整梯队。

其强，不在于板块大面积普涨，而在于**逆指数走强、逆情绪承压走强**。这类强势通常更有交易价值。

### 2. 算力
算力是与AI硬件并行的核心分支，板块净流入约 **46.88亿元**。早盘 **思特奇（300608）20cm涨停**，午后 **云赛智联（600602）直线封板**，并带动**中贝通信（603220）**等回流。催化上有“**普惠算力赋能中小企业发展专项行动**”的政策刺激，属于当日少数具备消息支持且盘中有二次强化的方向。

不过需要注意，算力板块指数涨跌幅为 **-1.29%**，说明资金更偏向**少数核心个股聚焦**，而非板块普遍性扩散。

## 轮动支线

### 1. 医药
医药延续强势，但更偏**情绪抱团**而非板块共振。核心是**津药药业（600488）六连板**、**重药控股（000950）二连板**。但板块整体涨跌幅为 **-2.5%**，净流出约 **33.01亿元**，说明这是典型的“**龙头强、板块弱**”。

### 2. 跨境支付
竞价阶段最强，开盘后迅速分化。**中油资本（000617）**、**翠微股份（603123）**竞价强势，但盘中出现**相继炸板**，最终仅翠微股份回封，**贝肯能源（002828）**则明显回落。板块净流入约 **20.41亿元**，但走势是**高开兑现型**，说明资金认可消息刺激，却不愿在弱市环境下持续抬升板块高度。

### 3. 电子气体 / GPU / 航运 / 化工
这些方向都出现过日内脉冲，但更接近**缩量弱市中的题材补位**：
- 电子气体相对独立，板块涨幅 **+1.46%**，有 **中船特气（688146）20cm**带动。
- GPU有资金尝试回流，净流入约 **12.80亿元**，但持续性仍待验证。
- 航运一度活跃，**招商南油（601975）**冲高，但收盘摘要显示其炸板，说明承接不强。
- 化工早盘异动，但板块跌幅 **-2.63%**、净流出约 **25.29亿元**，明显不具备持续主升特征。

## 负反馈方向

### 1. 商业航天
午后明显走弱，**神剑股份（002361）**、**中超控股（002471）**、**巨力索具（002342）**集体大跌。板块涨跌幅 **-1.57%**，即使有小幅净流入，也没有转化为赚钱效应，说明更多是存量博弈和高位兑现。

### 2. 算电协同 / 高位连板补跌
**新中港（605162）**炸板翻绿并逼近跌停，**广西能源（600310）**、**乐山电力（600644）**午后跌停，意味着前期高位题材的风险开始集中释放。

### 3. 高位人气股尾盘分歧扩大
尾盘**星辉环材（300834）**、**美诺华（603538）**、**舒华体育（605299）**持续回落，连同**联环药业（600513）**等跌停，说明资金对高位非主线品种的容忍度快速下降。

# 核心个股

市场当日的高辨识度个股，可以分为三类：

### 1. 主线带动型
- **汇源通信（000586）**：AI硬件方向前排连板，承担情绪锚点作用。
- **德科立（688205）**：20cm涨停，提供板块弹性，强化通信方向赚钱效应。
- **长飞光纤（601869）**：趋势中军，再创历史新高，对板块信心支撑最强。
- **亨通光电（600487）**：权重品种盘中触板，说明不仅是小票活跃，机构和大资金也有参与迹象。

### 2. 情绪抱团型
- **津药药业（600488）**：六连板，当前连板高度代表之一，是医药方向核心辨识度。
- **重药控股（000950）**：二连板，属于津药药业的低位补涨与板块助攻。

### 3. 分歧验证型
- **云赛智联（600602）**：午后直线涨停，是算力回流的关键确认点。
- **翠微股份（603123）**：跨境支付中少数完成回封的个股，说明该方向仍有局部活口。
- **新能泰山（000720）**：九天七板，属于高位抱团标杆，但其所在环境已经明显转弱，更偏情绪观察意义。
- **新中港（605162）**：炸板翻绿并逼近跌停，是高位亏钱效应的重要信号。
- **神剑股份（002361）**：商业航天午后跳水，带动板块负反馈扩散。

# 情绪判断

当前短线情绪更接近**退潮中的局部抱团**，而不是新周期启动。主要依据有三点：

1. **广度极差**：下跌家数4459家，远高于上涨家数698家，市场不是结构性普涨，而是少数热点维持活口。
2. **高位分歧加剧**：新中港、星辉环材、美诺华、舒华体育等尾盘走弱，高位股承接明显下降。
3. **主线与非主线分化明显**：AI硬件、算力尚能聚焦资金，但大量轮动支线冲高回落，说明市场偏好已收缩到少数高辨识度标的。

| 维度 | 判断 | 依据 |
|---|---|---|
| 情绪阶段 | 弱分歧向退潮侧倾斜 | 跌停24家，尾盘补跌增多 |
| 赚钱效应 | 局部存在 | 集中在AI硬件、算力、医药龙头 |
| 亏钱效应 | 明显扩散 | 高位炸板、连板股回落、超20家跌停 |
| 资金偏好 | 核心抱团、趋势中军、20cm弹性 | 长飞光纤、亨通光电、德科立、思特奇 |
| 核心矛盾 | 指数与情绪走弱，但科技主线仍有局部承接 | 弱市下主线能否继续独立走强 |

# 明日关注

明日观察重点应围绕“**主线能否穿越弱势环境**”展开，而不是盲目扩散到低辨识度轮动题材。

## 情绪锚点
- **津药药业（600488）**：连板高度能否继续维持，是短线情绪上限的重要参考。
- **新能泰山（000720）**：高位抱团是否继续稳住，决定高位股亏钱效应是否继续扩散。
- **新中港（605162）**：若继续弱势，可能强化高位退潮预期。

## 主线锚点
- **长飞光纤（601869）**、**亨通光电（600487）**：观察趋势中军是否继续稳住通信主线强度。
- **汇源通信（000586）**、**德科立（688205）**：观察前排连板与20cm弹性是否继续正反馈。
- **云赛智联（600602）**、**思特奇（300608）**：观察算力回流是一次性脉冲，还是能够接力通信方向形成共振。

## 风险点
- 市场成交额若继续低于 **1.7万亿** 且再缩量，题材持续性可能进一步下降。
- 若跌停家数继续维持高位或扩大，说明情绪退潮未结束。
- 医药、跨境支付若只剩单个龙头维持，板块跟风继续掉队，则需警惕抱团股补跌。
