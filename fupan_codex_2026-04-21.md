# 市场总览

4月21日市场走出一轮典型的“早盘承压、午前弱修复、午后权重带回指数”的盘口结构。早盘三大指数一度走弱，`09:39` 时深成指、创业板指跌超 `-1%`，`10:58` 跌幅收窄至近 `-0.5%`，到 `13:16` 三大指数已集体翻红，说明指数层面更多依赖大市值科技与新能源权重修复。

成交额方面，`09:42` 沪深两市成交突破 `5000亿`，`10:10` 突破 `1万亿`，`11:15` 突破 `1.5万亿`，盘中一致预期全天成交“不足 `2.5万亿`”，较前一交易日缩量约 `5%`。这意味着市场并非增量普涨，而是存量资金在高低切换和局部回流间反复博弈。

涨停家数方面，payload 未提供全市场精确统计，无法给出总量结论；但从时间线可确认，商业航天、算电协同、电子气体、影视院线、AI硬件分支、机器人等方向均有涨停出现。情绪层面则较清晰，属于**分化修复**而非一致转强，主要特征是高位负反馈先出现，随后权重和部分辨识度品种带动局部回流。

# 主线与轮动

## 最强主线

从全天结构看，**AI硬件/通信链仍是最强主线，只是日内以“先杀后回流”的方式演绎**。早盘 `09:26-09:43`，英维克一字跌停，天孚通信、胜宏科技、长飞光纤、新易盛等偏弱，属于高位科技方向集中释放压力；但到 `11:02` 开始，东山精密、剑桥科技率先修复，`13:06` 致尚科技触及 `20cm` 涨停、东山精密涨超 `7%` 再创历史新高，`13:16` 又与兆易创新、宁德时代共同推动指数翻红。

需要注意的是，板块层面“通信”当日涨跌幅仍为 `-0.56%`、净流入为 `-88.47亿` 左右，说明**回流更多集中在少数高辨识度个股，并未形成板块级全面回暖**。因此，这条线当前仍是主线，但更偏“核心抱团”，并非普遍扩散。

## 轮动支线

### 1. 商业航天
商业航天是早盘最先强化的情绪方向。`09:25` 顺灏股份首板涨停，`09:27` 竞价即异动，顺灏股份大单一字，博云新材、神剑股份高开，`09:36` 神剑股份涨停、走出五天三板。题材催化来自轨道辰光完成 PreA1 轮融资。  
但问题在于，板块涨跌幅仅 `-0.76%`、净流入为负，且博云新材后续翻绿，说明**题材有情绪辨识度，但板块承接并不完整，更像局部点火而非主升扩散**。

### 2. 算电协同
算电协同是上午后半段最明确的轮动方向。`10:26` 粤电力A直线涨停，随后 `11:26-11:28` 华电辽能、浙江新能相继涨停，晋控电力、宁波能源跟随走强。这个方向的优势在于位置相对不高，且在科技主线分歧时具备承接部分风险偏好资金的能力。  
不过数据同样没有给出独立板块涨跌幅与净流入，强度判断仍应停留在“日内轮动有效”，暂不足以确认接力为新主线。

### 3. 电子气体
电子气体是当天少数**板块涨幅与净流入同时为正**的方向。板块涨幅 `+3.31%`，净流入约 `3.42亿`，`10:06` 华特气体触及 `20cm`，金宏气体涨超 `11%`。这说明资金在科技大票承压阶段，尝试向半导体材料细分切换。  
但该方向从时间线看更多是脉冲式强化，后续持续发酵数据不足，仍按轮动支线看待更合适。

### 4. 锂电池
锂电池在 `09:46` 由天齐锂业带动异动，板块净流入约 `25.77亿`，是给定数据中少数净流入较高的方向；但板块涨跌幅仅 `-0.07%`，且 `11:30` 明确提到“天齐锂业冲高回落”。这说明**资金有权重试盘动作，但没有形成有效的板块趋势强化**，更多还是指数修复过程中的权重辅助线。

### 5. 影视院线、医药
影视院线与医药均有早盘异动。影视院线板块涨幅 `+0.94%`、净流入约 `4738万`，北京文化首板，属于典型防守型或事件驱动型轮动。医药虽有益佰制药直线涨停，但板块涨跌幅 `-0.99%`、净流入约 `-31.47亿`，跟随不足，持续性存疑。

# 情绪判断

当前情绪更接近**分化期中的修复段**，不是新一轮一致高潮。

一方面，负反馈很明确。圣阳股份、英维克直接跌停，博云新材高开回落，说明前期高位方向开始出现淘汰赛，追高资金承受明显回撤。另一方面，市场并未出现系统性退潮，因为东山精密、剑桥科技、致尚科技、顺灏股份、华特气体等高辨识度个股仍能反复被资金选择，指数也在午后被权重拉回红盘。

如果用阶段划分，今天更像是：

| 维度 | 判断 |
| --- | --- |
| 短线情绪阶段 | 分化修复 |
| 主线状态 | 未结束，但由普涨转向核心抱团 |
| 轮动状态 | 高频切换，支线持续性偏弱 |
| 操作难度 | 明显提升，后排与跟风容错率低 |

核心矛盾在于：**高位科技方向已经开始释放筹码压力，但市场又找不到足以完全替代AI硬件的新主线，因此资金只能在主线核心、低位轮动和指数权重之间来回切换。**

对应的资金偏好也比较清楚：

1. 偏好有辨识度、能承接资金回流的主线中军与趋势核心。
2. 偏好有事件催化、位置相对低的轮动支线前排。
3. 回避高位跟风、纯补涨和早盘冲高后失去承接的品种。
