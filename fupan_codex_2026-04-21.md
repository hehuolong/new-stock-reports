# 市场总览

4月21日，指数层面呈现明显的“早弱午修复”节奏。早盘三大指数一度下挫，深成指、创业板指跌超1%；午后在东山精密、宁德时代、兆易创新、中际旭创等权重带动下，三大指数集体翻红。  
成交额方面，盘中14:05沪深两市成交突破2万亿，且直播明确提示“预计全天成交不足2.5万亿”，较上个交易日缩量约6%，说明指数修复更多依赖权重抱团，而非普涨增量。

涨停家数方面，给定数据未提供全市场精确统计，无法机械给出总数；但从事件流看，日内涨停仍有一定密度，集中在商业航天、算电协同、局部AI硬件、电子气体、锂电等方向。  
市场情绪方面，核心特征不是冰点，而是**高低切下的分化修复**：一边是圣阳股份、英维克等高位/前强方向出现跌停或大幅负反馈；另一边是商业航天、算电协同、局部AI硬件回流品种承接活跃，但整体下跌家数长期维持在3000家至3500家以上，说明赚钱效应并不扩散。

# 主线与轮动

## 最强主线：商业航天

商业航天是日内辨识度最高、持续性最完整的方向。竞价阶段即有异动，顺灏股份一字首板，神剑股份快速涨停并在尾盘回封至五天三板，巨力索具尾盘首板涨停，中国卫星在竞价龙头观察名单中亦有较高辨识度。板块信号显示，商业航天板块涨跌幅为-0.34%，但净流入为4.48亿元，这说明虽然板块整体并非强趋势普涨，但**核心票获得了集中抱团**。

从催化看，已知驱动来自“轨道辰光完成PreA1轮融资”的消息发酵。  
从结构看，这条线的优势不在板块面整体强度，而在于**辨识度个股足够集中，且有连板高度支撑**。不足在于，博云新材高开回落翻绿，说明板块内部也存在明显分歧，暂时更偏“核心抱团”，不是全面扩散。

## 轮动支线

### 1. AI硬件 / 通信：先杀后修复，权重强于题材

AI硬件是全天最值得重视的“修复线”。早盘英维克一字跌停，天孚通信、胜宏科技、长飞光纤、新易盛等承压，明显属于前期热门方向的兑现压力；但10:58后东山精密走强，11:02剑桥科技翻红并创趋势新高，13:06致尚科技20cm涨停，东山精密涨超8%并创历史新高，午后成为带动指数翻红的重要力量。

需要注意的是，板块信号里通信板块仍是**净流出71.70亿元、板块涨跌幅-0.2%**，这意味着这里并非真正意义上的板块反转，而是**资金从高位拥挤品种撤离后，向更有辨识度、更偏趋势中军的个股做再集中**。  
换言之，AI硬件今天体现的是“结构修复”，不是“全面修复”。

### 2. 算电协同：弱市里的稳定轮动支线

10:26粤电力A直线涨停，随后浙江新能、华电辽能、晋控电力、宁波能源等持续强化，到11:28已经形成较完整的板块联动。  
这一方向的特点是：

- 启动时间早，承接市场早盘风险释放后的防守需求。
- 个股联动相对整齐，涨停数量较集中。
- 与算力主线存在一定映射关系，偏“基础设施配套”的逻辑补涨。

但数据未提供该板块整体净流入和涨跌幅，无法进一步确认其是否具备跨日成为主线的强度。目前更适合作为轮动加强项观察。

### 3. 锂电池：有资金尝试，但板块强度一般

9:46天齐锂业一度逼近涨停，板块净流入17.95亿元，是已给数据中少数净流入较显著的方向；午后英联股份涨停，宁德时代也维持强势震荡。  
但从直播描述看，天齐锂业“冲高回落”，说明资金态度更接近事件驱动下的短打轮动，持续性仍待确认。若后续不能形成更多涨停或中军趋势共振，这里大概率还是支线。

### 4. 电子气体、影视院线、医药、机器人：典型弱市快轮动

电子气体中华特气体20cm涨停、金宏气体大涨，属于高弹性分支突袭；影视院线有北京文化首板；医药有益佰制药异动；机器人午后日盈电子直线拉板但冲板回落。  
这些方向共同特征是：

- 有脉冲，但持续时间短。
- 缺少跨时段反复强化。
- 对指数和情绪的带动有限。

因此更适合归类为**弱市里的短线轮动噪声**，而非当前核心。

## 负反馈方向：算力高位股与前强科技股

今天最明确的负反馈来自两组资产：

### 1. 算力高位连板股
算力板块整体跌幅-0.9%，净流出高达105.07亿元，是数据中负流出最明显的方向。圣阳股份跌停，利通电子、宏景科技大跌，华升股份盘中炸板且最终数据表现不佳。虽然有工信部关于“普惠算力赋能中小企业”的政策表述，但盘面体现为**利好不涨、筹码兑现**，这是典型的高位退潮信号。

### 2. AI硬件前强权重 / 核心抱团分裂
英维克一字跌停，天孚通信、胜宏科技、长飞光纤等早盘承压，说明此前拥挤交易的方向开始出现内部切换。只是午后东山精密、剑桥科技、致尚科技等完成回流，才让整体观感没有进一步恶化。  
这意味着当前科技风格并未彻底结束，但内部已经从“全面做多”切换到“只做最强辨识度”。

# 情绪判断

今天更接近**分歧后的弱修复**，不是一致性回暖。

| 观察维度 | 表现 | 判断 |
|---|---|---|
| 高位股反馈 | 圣阳股份跌停，博云新材回落，沃格光电冲高回落 | 高位接力仍偏谨慎 |
| 中军趋势 | 东山精密、宁德时代、兆易创新等走强 | 大资金偏好低波动、高辨识度中军 |
| 板块扩散 | 多数方向仅脉冲轮动 | 题材持续性不足 |
| 市场宽度 | 下跌家数长期在3000家至3500家以上 | 个股情绪弱于指数表现 |

因此，当前情绪阶段可定义为：**退潮末端到修复初期之间，但修复主要集中在少数核心资产，尚未完成情绪面普遍修复。**

市场核心矛盾也比较清晰：

- 一方面，指数有权重和趋势中军托底。
- 另一方面，短线高位接力和题材普涨能力明显不足。

资金偏好则体现为：

- 回避高位拥挤、纯情绪博弈品种。
- 优先选择有板块辨识度的核心票。
- 偏好“趋势中军 + 弹性先锋”的组合，而不是全面铺开。
