# 市场总览

4月23日市场呈现典型的“放量分歧”结构。指数层面，数据仅明确显示三大指数高开后持续走弱，至11:30深成指、创业板指跌超1.5%，且10:43仍在创日内新低；上证具体涨跌幅未给出，无法做更精确判断。成交额显著放大，9:58突破1万亿、10:34突破1.5万亿、13:04突破2万亿，均较前一交易日放量18%至22%，盘中一致预期全天成交将超过3万亿，说明资金活跃度很高，但并未转化为普涨。

涨停家数方面，原始数据未提供全市场精确统计，不能臆造；但从时间线可确认，盘中涨停分布在AI硬件、算力、商业航天、燃气轮机、化工、白酒、医药、石油石化、算电协同等多个方向。与之对应，11:30明确提到“超10家个股跌停”，且下跌家数从开盘后近3500家扩大至超4000家，说明情绪并非强修复，而是高位分化、局部抱团。

整体看，市场情绪处于**高成交下的分歧加剧阶段**：有连板龙头维持高度，也有大量跟风和高位股承压，资金风格偏向少数高辨识度核心票与事件驱动分支，而非全面扩散。

| 维度 | 盘面特征 | 交易解读 |
| --- | --- | --- |
| 指数 | 高开低走，深成指/创业板指跌超1.5% | 赚钱效应弱于成交热度 |
| 成交额 | 13:04已超2万亿，预计全天超3万亿 | 资金活跃，但主要体现在高换手博弈 |

| 情绪 | 连板与趋势核心仍有表现，但跌停超10家 | 高位分歧，非一致性强势周期 |

# 主线与轮动

## 最强主线

从辨识度、持续性和资金记忆点看，**泛AI链条仍是市场最强主线，但内部已经明显分层**，可拆为“AI硬件/通信”和“算力”两部分。

1. **AI硬件/通信**
   早盘最强开局之一，9:31中际旭创总市值突破万亿，9:38云南锗业被定义为五天四板、杭电股份二连板、中际旭创再创历史新高，说明开盘资金仍在沿着最熟悉的科技主线定价。
   
   但问题也最早出在这里。9:54通信方向即出现明显分化，云南锗业、杭电股份炸板，天孚通信跳水跌超4%，新易盛跟跌；10:43天孚通信跌超8%，中际旭创翻绿，板块内部高位筹码松动。直到13:20、13:22，汇绿生态首板涨停，中际旭创拉回均线、天孚通信跌幅收窄，才算出现局部修复信号。

   需要注意的是，板块信号里通信板块涨跌幅为-2.13%，净流出约232.47亿元，说明即便午后有回流，仍属于**大分歧后的局部修补**，并不是重新转强。

2. **算力**
   9:35算力活跃，铜牛信息涨超14%，协创数据被描述为“再创历史新高”，同时有GPU租赁价格上涨的外部催化。10:30和11:30又继续出现浙文互联、莲花控股首板，说明算力方向承接了科技资金的部分扩散需求。

   但板块层面，算力涨跌幅为-1.43%，净流出约33.48亿元，说明其强更多体现为**个股活跃而非板块强势**。换句话说，算力今天更像主线内部的活口，而不是能独立带领指数和情绪修复的绝对核心。

## 负反馈方向

1. **锂电池**
   9:46即明确走弱，西藏珠峰跌停，海科新源跌超10%，板块跌1.76%，净流出约109.53亿元，是日内最明确的负反馈赛道之一。这个方向既没有承接，也没有修复描述，属于被资金显著回避。

2. **AI硬件高位跟风**
   虽然主线仍在，但高位跟风票和部分权重承压明显。云南锗业炸板回落，天孚通信大跌，沃格光电深水震荡，这意味着资金只愿意集中在少数核心资产，不愿意为板块内普遍溢价买单。

3. **微盘股与非核心高位股**
   9:44微盘股指跌超1%，11:30又有“超10家跌停”，说明情绪负反馈主要集中在**辨识度不够、缺乏板块地位或纯情绪透支**的标的上。

# 核心个股

市场当前的核心矛盾，不是“有没有热点”，而是**资金在放量环境下，选择继续聚焦高辨识度核心，还是向板块内部扩散**。从结果看，扩散失败更明显，资金偏好集中在以下几类：

1. **有产业趋势支撑的科技权重**
   - `300308 中际旭创`：AI硬件权重锚，早盘总市值破万亿、再创历史新高，盘中虽有翻绿和回落，但午后重新拉回均线，是科技主线的容量风向标。
   - `300394 天孚通信`：负反馈锚。其大跌直接放大了通信方向的分歧程度，属于高位情绪释放口。
   - `300895 铜牛信息`：算力弹性票，体现资金更愿意做主线中的高弹性进攻。

2. **高连板情绪核心**
   - `002081 金螳螂`：五连板，是目前连板高度的重要情绪锚。
   - `603017 中衡设计`：四天三板，作为商业航天链补充核心，强化了题材辨识度。
   - `603318 水发燃气`：四天三板，燃气轮机方向核心，体现事件驱动支线仍可承接短线资金。

3. **午后新强化或修复载体**
   - `000539 粤电力A`：三天二板，午后算电协同代表票。
   - `600726 华电能源`：算电协同首板，强化“电力配套服务AI”的资金偏好。
   - `001267 汇绿生态`：AI硬件午后回流的首板先锋，代表板块在分歧后尝试找新面孔修复。

从带动关系看，比较清晰的链条有三条：

- **中际旭创 -> AI硬件权重稳定 -> 午后汇绿生态等回流**
- **金螳螂 -> 中衡设计/中国卫星 -> 商业航天维持热度**
- **华电能源 -> 粤电力A -> 算电协同从跟随走向独立观察方向**

# 情绪判断

当前短线情绪更接近**分歧中后段的结构性抱团**，而非退潮末端，也不是新一轮一致加速。

| 维度 | 表现 | 判断 |
| --- | --- | --- |
| 连板高度 | 金螳螂五连板，水发燃气四天三板，中衡设计四天三板 | 高度仍在，情绪未崩 |
| 跌停与炸板 | 超10家跌停，云南锗业炸板，天孚通信大跌 | 亏钱效应扩散 |
| 主线状态 | AI硬件/算力有回流但分化剧烈 | 主线未死，但难言顺畅 |
| 轮动效率 | 白酒、医药、石油、化工等多方向轮动 | 资金缺乏统一进攻方向 |
| 市场环境 | 放量、下跌家数超4000 | 交易难度较高 |

更具体地说，市场正在经历两类博弈：

1. **高辨识度核心是否继续获得溢价**
   如果继续集中在金螳螂、中际旭创、粤电力A这类票，说明抱团还能延续。
2. **主线内部能否从“核心独强”切换到“板块修复”**
   如果AI硬件只能靠中际旭创孤军支撑，而天孚通信、云南锗业等持续承压，则主线只剩容量抱团，短线空间会受限。

# 明日关注

明日观察重点，建议聚焦“情绪锚点、主线锚点、风险点”三条线。

1. **情绪锚点**
   - `002081 金螳螂`：连板高度能否继续打开，直接影响短线接力情绪。
   - `603318 水发燃气`：若高位支线核心补跌，说明情绪抱团开始松动。
   - `000539 粤电力A`：午后强化方向能否获得溢价，决定新分支是否具备接力价值。

2. **主线锚点**
   - `300308 中际旭创`：是否能继续稳住均线并带动通信/AI硬件修复，这是主线是否还有组织性的关键。
   - `300394 天孚通信`：若继续大跌，会压制整个AI硬件风险偏好；若止跌，则主线分歧有望收敛。
   - `001267 汇绿生态`、`603618 杭电股份`：看主线能否从权重抱团扩展到弹性票修复。

3. **风险点**
   - 市场虽放量，但下跌家数超4000家，说明量能并未改善普遍赚钱效应。
   - 通信板块净流出约232.47亿元，锂电池净流出约109.53亿元，赛道资金撤离压力仍大。
   - 若明日核心高辨识度个股不能继续超预期，而负反馈方向继续扩散，市场可能从“结构分歧”转向“高位退潮”。

结论上，当前盘面最核心的矛盾是：**成交极度活跃，但赚钱效应高度集中，资金只认可少数核心，不接受普遍扩散。** 操作上应更重视锚点反馈，而不是看到轮动就追逐分支。
