# 主线与轮动

## 最强主线

早盘最强主线仍是 **AI硬件+算力**，但定性上更准确地说，是 **主线地位仍在，内部已经进入高位分歧**。

核心依据：
- 9:31，**300308 中际旭创** 盘中被反复强调，总市值突破万亿，具备绝对辨识度。
- 9:35，算力继续活跃，**300895 铜牛信息** 涨超12%，**301396 宏景科技** 跟涨。
- 9:38，AI硬件延续强势，**002428 云南锗业** 五天四板，**603618 杭电股份** 二连板，**002281 光迅科技** 大涨。
- 但到 9:54，AI硬件出现明显分化，**云南锗业、杭电股份炸板，300394 天孚通信跳水超5%，300502 新易盛跟跌**。

这说明主线没有被替代，但交易难度上升，资金从“无脑追核心”转向“只认辨识度、只做最强分支”。

# 情绪判断

当前情绪更接近 **“高位主线分歧期中的局部轮动”**，而不是全面退潮，也不是一致性主升。

| 观察项 | 判断 |
| --- | --- |
| 情绪阶段 | 分歧期 |
| 资金行为 | 一边留恋AI主线核心，一边尝试切防御和事件驱动新分支 |
| 市场结构 | 放量但不普涨，典型结构性行情 |
| 交易难点 | 后排与跟风股承接不足，炸板和跳水增多 |
| 资金偏好 | 偏好高辨识度龙头、趋势中军、事件催化前排，不愿意大面积做扩散 |

市场核心矛盾在于：
1. **成交活跃，但宽度很差**，说明有资金在交易，但多数个股并未受益。
2. **AI硬件仍是最强主线，但高位分歧开始释放**，资金必须在“继续抱核心”与“切新方向”之间做选择。
3. **新轮动方向有动作，但板块级别强度不足**，多数还停留在前排个股强、板块整体一般的阶段。
