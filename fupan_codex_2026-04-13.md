# 主线与轮动

## 最强主线

从辨识度和连板高度看，当前最强主线更接近**AI硬件/算力链的局部强化**。

一方面，09:15竞价阶段明确点名了AI硬件与算力方向，涉及**600103 青山纸业、600345 长江通信、002384 东山精密、000889 中嘉博创**；另一方面，09:25 **002364 中恒电气涨停并晋级三连板**，这是已给数据里最清晰的高度标。短线交易里，高度板本身就是资金风向标，中恒电气的三连板意义大于一般首板，它说明资金在弱指数背景下，仍愿意给算力链辨识度溢价。

但要注意，算力链内部也不是全面走强：**000889 中嘉博创**竞价为负，说明资金聚焦的是高辨识度个股，而非板块一致性扩散。这意味着该方向现阶段更像**龙头抱团**，还未看到完整板块共振证据。

## 轮动支线

这是开盘最明确的事件催化方向。  
09:28市场动态显示，**平潭发展、厦门港务、漳州发展**集体高开；板块信号给出“海峡两岸”板块净流入约**591.95万元**，催化来自**4月12日发布的十项促进两岸交流合作政策措施**。

# 核心个股

当前高辨识度个股可以分成三类：

这是已知数据中的高度核心。三连板意味着它不仅是算力方向的强度标，也承担着短线情绪锚的功能。如果后续继续超预期，容易带动算力链补涨；若高位分歧明显，则对短线连板接力构成压力。

## 2. 事件驱动前排

**002439 启明星辰**、**002297 博云新材**

## 4. 潜在方向辨识股

这组三只来自竞价中的AI硬件线索。青山纸业竞价最强，长江通信、东山精密则体现一定跟随性。如果日内AI硬件获得回流，这组票会是重要映射对象。

# 情绪判断

当前更接近**弱指数背景下的局部活口行情**，而不是全面修复。

一是指数低开、下跌家数偏多，说明大环境不支持无差别做多。  
二是最强反馈集中在少数个股：中恒电气的连板、福建题材前排高开、业绩首板启动，这些都体现为**点状机会**。  
三是从板块一致性看，海峡两岸方向虽然有政策催化，但板块整体仍为负；算力虽然有中恒电气三连板，但同链条个股表现并不整齐。这说明情绪尚未进入强一致主升阶段，更像是**分歧市中的结构性博弈**。

| 维度 | 判断 | 依据 |
|---|---|---|
| 情绪阶段 | 分歧偏谨慎 | 指数低开、下跌个股超3500家 |
| 赚钱效应 | 局部存在 | 连板股、事件驱动股、业绩首板有表现 |
| 亏钱效应 | 背景性存在 | 大盘普跌，非核心方向承接不足 |
| 资金偏好 | 高辨识度、强催化、可验证逻辑 | 三连板、政策刺激、业绩增长 |
| 适合策略 | 聚焦前排、少做后排扩散 | 板块一致性不足，容错率偏低 |

市场核心矛盾在于：**指数与市场宽度偏弱，但局部题材和高辨识度个股又维持活跃，资金只能在防守背景下寻找高胜率切口。**  
对应的资金偏好很清楚：

1. 偏好**有高度的连板核心**，如中恒电气。
2. 偏好**有明确催化的事件题材前排**，如福建方向。
3. 偏好**有基本面支撑的业绩首板**，如启明星辰、博云新材。
4. 回避**缺乏领涨核心、仅停留在概念层面的后排扩散**。

1. **指数继续走弱的系统性风险**

3. **追高风险偏大**
