# 市场总览

4月13日早盘，三大指数呈现**低开修复、随后小幅回落**的节奏，创业板指相对更强，一度涨近1%，截至午间口径仍涨近0.5%。成交层面，10:24两市成交已突破1万亿，预计全天成交约**2.1万亿**，较前一交易日**缩量约6%**，说明市场仍有交易活跃度，但增量资金并未继续放大。

短线赚钱效应并不差，但市场宽度偏弱。数据明确指出，早盘**下跌个股一度超3500家，午间仍超3000家**，体现出指数修复主要依赖部分权重和前排题材，个股层面仍是典型的结构性行情。  
涨停家数方面，payload未给出完整统计，只能确认**涨停个股主要集中在AI硬件、算力、玻纤、部分业绩股与高位连板股**，无法据此精确还原全市场涨停家数。

# 主线与轮动

## 最强主线

### 1. AI硬件 / 算力链
这是早盘最明确的资金主攻方向，且内部形成了“**权重趋势 + 前排连板 + 分支补涨**”的较完整结构。

一方面，AI硬件方向有**东山精密（002384）创历史新高**、**瑞斯康达（603803）首板**、**华盛昌（002980）首板**、**科瑞技术（002957）五天三板**，说明资金既做辨识度，也做高低切。另一方面，算力方向有**中恒电气（002364）三连板**、**圣阳股份（002580）三连板**、**中安科（600654）反包涨停**、**润建股份（002929）四天二板**，叠加**海光信息（688041）涨超6%**、**万通发展（600246）涨停**，主线强度和层次感都更完整。

从板块信号看，**算力板块净流入约16.49亿元**，且明确有催化信息“DeepSeek V4实现与华为昇腾等国产芯片全栈深度适配”，因此这条线不只是情绪博弈，也带有一定事件驱动属性。

### 2. AI硬件分支中的玻纤
玻纤是今日早盘强度最高的细分之一。板块涨幅**+5.78%**，净流入约**17.49亿元**，在给定板块信号中属于最强。  
个股上，**宏和科技（603256）涨停且创历史新高**，**国际复材（301526）涨15.73%**，**中国巨石（600176）涨停**。这说明资金不只是在做小票情绪，而是在做产业链分支强化，且有趋势龙头与容量票共振。

## 轮动支线

### 1. 锂电池 / 储能
早盘锂电、储能都有脉冲，但更像**指数修复阶段的轮动承接**，而不是资金一致性的新主升。

锂电池方向里，**天华新能（300390）涨超8%创趋势新高**，**宁德时代（300750）走强**，但板块净流入约**-23.94亿元**；储能方向里，**德业股份（605117）逼近涨停**、**固德威（688390）走强**，但板块净流入约**-23.70亿元**。也就是说，盘面有表现，资金面却未形成有效净流入，更多体现为局部龙头和权重带动下的轮动，而非主攻。

### 2. 存储
存储方向延续活跃，**德明利（001309）再创历史新高**，**佰维存储（688525）**、**江波龙（301308）**跟涨，板块涨幅**+0.56%**。  
但该方向板块净流入约**-5.23亿元**，说明其强度更多偏趋势抱团，而不是全面扩散。

### 3. 次新股
次新股有明显弹性，**C三瑞（301696）涨超28%**、**C红板（603459）涨近18%**、**纳百川（301667）涨近14%**。  
不过次新股的特征是高波动、高弹性，更多是风险偏好回升时的边际受益方向，暂时难定义为市场核心主线。

### 4. 化工、商业航天、海峡两岸
这几个方向都出现过盘中异动，但持续性和板块合力均较弱。  
其中化工板块净流入约**8.23亿元**，但板块涨跌幅是**-0.28%**；商业航天净流入约**7.89亿元**，但个股多为跟涨；海峡两岸概念更多停留在竞价和开盘异动，后续并未看到强化。整体看，均偏消息脉冲式轮动。

## 负反馈方向

负反馈主要来自两类：

### 1. AI硬件内部的分化
虽然AI硬件整体仍强，但开盘即出现明显分歧。  
例如**青山纸业（600103）高开后大幅回落**，**长飞光纤（601869）水下震荡且跌幅较大**，说明该方向并非全面一致上攻，而是资金更偏好“有辨识度、有新高属性、能封板”的核心票，后排和弱辨识度标的承压明显。

### 2. 竞价热门但开盘承接不足的方向
如竞价端提及的**中嘉博创（000889）**、**彩虹股份（600707）**等，盘中表现并不强，说明资金在缩量背景下更强调兑现效率和辨识度，泛题材跟风难以获得持续承接。

# 情绪判断

当前市场更接近**分歧中的局部强修复**，而非全面情绪主升。

| 维度 | 现状 | 判断 |
|---|---|---|
| 指数 | 低开修复，创业板偏强 | 指数层面偏稳 |
| 宽度 | 下跌家数仍超3000 | 个股赚钱效应不全面 |
| 连板 | 华远控股五连板，中恒电气/圣阳股份三连板 | 短线情绪并未退潮 |
| 反包 | 中安科、华电辽能反包涨停 | 高位股修复意愿较强 |
| 成交 | 预计2.1万亿，缩量6% | 存量博弈特征明显 |
| 风格 | 前排抱团、后排分化 | 只适合做核心，不适合大面积扩散博弈 |

综合来看，情绪阶段可定义为：**存量环境下的结构性活跃阶段**。  
资金偏好非常清晰：

1. 偏好**有产业逻辑加持的科技主线**，尤其是算力、国产算力、AI硬件分支。
2. 偏好**高辨识度个股**，包括连板、反包、历史新高和容量中军。
3. 不愿为后排跟风、竞价脉冲题材和无持续性的消息股提供过多溢价。

市场核心矛盾在于：**指数修复与个股普跌并存，说明资金只在少数确定性方向聚焦，广谱风险偏好并未真正打开。**

## 风险点
1. 成交量较前一日缩量6%，若明日继续缩量，主线延续性会受压制。
2. 当前下跌个股仍超3000家，说明一旦核心票松动，亏钱效应可能快速扩散。
3. 锂电、储能、化工、商业航天等支线虽有轮动，但资金净流入与板块涨幅并不完全匹配，追高性价比偏低。
