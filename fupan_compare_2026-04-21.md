# 延续与变化

## 1. 市场环境
上一交易日的核心特征是放量、宽度尚可、题材轮动维持强度；当前交易日则是缩量修复、下跌家数长期维持在3000家至3500家以上，说明指数翻红并未带来情绪普遍改善。

这意味着：
- `被延续的判断`：市场仍不是指数单边主导，核心仍在题材与情绪结构。
- `被证伪的判断`：4月20日“增量资金支撑多题材并行活跃”的环境未延续，4月21日已经回到缩量背景，题材普涨基础减弱。
- `超预期变化`：指数午后翻红，但个股情绪没有同步修复，权重与题材明显脱节，这种“指数修复强于赚钱效应”的背离比上一日更突出。

## 2. 最强主线
商业航天连续两日都是最强主线，这一点被确认。  
但4月20日的强，表现为`博云新材六连板 + 中国卫星/中国卫通涨停 + 广联航空/天银机电弹性扩散`，属于高度、中军、弹性较完整的板块共振。  
4月21日的强，则表现为`顺灏股份竞价表态 + 神剑股份回封 + 巨力索具尾盘助攻`，更偏向少数核心票维持辨识度，板块整体并不强趋势。

结论是：
- `被延续的判断`：商业航天确实从“新主线候选”升级为“连续两日最强主线”。
- `被证伪的判断`：上一日对“中军扩容、创业板弹性同步维持”的预期没有完全兑现，博云新材回落翻绿，双卫星未进入当前核心观察票，说明主线从扩容转为收缩。
- `超预期变化`：神剑股份取代博云新材，成为更清晰的主线情绪锚，主线核心发生了内部换手切换。

## 3. AI链条与科技风格
4月20日的AI链仍是市场底仓，算力和AI硬件都有前排强标支撑，虽然已有20cm和高弹性方向分歧，但整体未坏。  
4月21日则出现明确分化：
- 算力高位股成为最强负反馈区，圣阳股份跌停，算力板块净流出高达105.07亿元。
- AI硬件没有全面转弱，而是从拥挤高位票出清后，转向东山精密、剑桥科技、致尚科技等趋势与弹性结合的修复结构。

因此：
- `被延续的判断`：4月20日提出“资金开始压缩无基本面支撑的追涨空间”“只做更高确定性的核心”，这一点在4月21日得到强化验证。
- `被证伪的判断`：上一日对圣阳股份、天通股份、通鼎互联继续承担AI前排强度的观察逻辑，至少在当前日没有延续成正反馈；尤其圣阳股份从情绪总龙头转为跌停负反馈锚点，属于明确证伪。
- `超预期变化`：AI硬件没有随算力一起整体退潮，而是出现“前强杀跌、趋势中军修复”的内部切换，东山精密成为指数修复抓手，强于单纯反抽预期。

## 4. 情绪结构
4月20日的情绪定义是“高位强势期中后段分歧”；4月21日则进一步下移到“退潮末端到修复初期之间”。  
两日共同点是都强调`去弱留强、核心集中`，不同点是4月21日的高位负反馈已经从局部现象变成主导矛盾之一。

具体看：
- `被延续的判断`：20cm、高位拥挤、纯情绪接力方向的风险继续暴露。
- `被证伪的判断`：上一日“高位连板仍有辨识度，主板接力未断”的环境强度没有继续维持到当前日，至少高位接力容错率显著下降。
- `超预期变化`：高位股退潮并未直接拖垮全部科技风格，反而促成资金向“趋势中军+高辨识度弹性”再集中，说明资金不是单纯撤退，而是在重构偏好。
