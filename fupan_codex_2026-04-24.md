# 主线与轮动

### 1. 芯片 / CPU
这是日内最清晰的主攻方向之一，特点是 **有外部催化、有权重承接、有板块净流入**。

- **催化**：英特尔给出强劲营收展望，盘后大涨，带动 CPU/芯片链条竞价异动。
- **资金表现**：
  - CPU 板块涨幅 **+3.22%**，净流入 **23.93亿元**
  - 芯片板块净流入 **43.17亿元**
- **核心表现**：
  - `600770 综艺股份` 二连板，且盘中走出 **T字板**，承担短线情绪锚作用
  - `688041 海光信息` 高开后承接强，盘中拉回均线并涨超8%
  - `688981 中芯国际`、`688347 华虹公司`、`688820 C盛合` 共同强化板块强度

交易上看，这一方向的优势不只是题材活跃，而是 **从连板到科创权重再到大市值核心形成了梯队**，比单纯题材脉冲更具持续性。

### 2. 锂电池
锂电池属于盘中强化最明显的轮动主攻方向，**板块净流入居前，但板块涨幅并不大**，说明更偏向资金做局部弹性与事件驱动，而非全面趋势反转。

- **板块净流入**：**66.80亿元**
- **代表个股**：
  - `000546 金圆股份` 直线涨停
  - `002176 江特电机` 首板涨停
  - `002812 恩捷股份` 首板涨停
  - `300390 天华新能` 涨超11%
  - `002192 融捷股份`、`002240 盛新锂能` 跟涨

从交易结构看，锂电更像是 **弱指数环境下的资金切换方向**，板块内有容量、有弹性，也有首板扩散，但当前数据中 **缺少更明确的产业催化**，因此持续性仍需次日确认。

# 情绪判断

当前市场更接近 **分化退潮中的局部试错期**，而不是全面修复期。

| 维度 | 当前状态 | 说明 |
|---|---|---|
| 指数情绪 | 偏弱 | 指数低开低走，创业板承压更重 |
| 短线接力 | 分化 | 苏州高新、水发燃气维持高度，但金螳螂跌停 |
| 趋势抱团 | 松动 | AI硬件权重大跌，资金开始从高位拥挤方向撤离 |
| 新主线试探 | 存在 | 芯片最强，锂电次强，昇腾和DeepSeek为补充 |
| 情绪阶段 | 弱修复中的结构博弈 | 有活口，但没有全面共振 |

市场核心矛盾在于：

1. **指数和市场宽度偏弱**，决定了多数题材难以全面扩散。
2. **高位AI硬件/算力资金出逃**，需要新的容量方向承接。
3. **芯片具备外部催化和权重承接**，是目前最像“新承接主线”的方向，但是否能从日内强转为阶段强，仍需后续验证。
4. **锂电、DeepSeek、华为昇腾等方向更多是轮动受益**，持续性要看次日是否能继续扩散到中军和连板结构。

资金偏好非常明确：

- 回避高位拥挤的 AI 硬件、算力
- 偏好有催化、位置相对合理、容量足够的科技分支
- 在弱市中更愿意做 **龙头、首板、辨识度核心**，而不是普遍撒网
